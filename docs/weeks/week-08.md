## 出題傾向と対策

中間試験では以下のようなテーマが出題されます。各週の内容を復習しましょう。

### 1. デジタル計測器の分解能（第6週）

::: tip 対策ポイント
- ビット数から分割数を計算する（$2^n$ 分割）
- 1ステップあたりの電圧値を求める
- 電圧値からデジタル出力値（整数値）への変換ができるようにする
:::

**練習問題**：10ビット分解能で 0〜5 [V] を計測する場合、計測電圧値が 1.7 [V] のときのデジタル出力値は？

<details>
<summary>解答</summary>

$$\Delta V = \frac{5}{2^{10}} = \frac{5}{1024} = 0.00488\dots \text{ [V]}$$

$$\text{デジタル値} = \left\lfloor \frac{1.7}{0.00488} \right\rfloor = \lfloor 348.16 \rfloor = 348$$

</details>

### 2. 補正と改ざん（第2週）

::: tip 対策ポイント
- **補正**：ずれの原因が特定できている場合に、そのずれ量だけ計測値を修正する手続き
- **改ざん**：ずれの原因がわからないのに、意図的にデータを書き換える行為
- 両者の違いを明確に説明できるようにする
:::

### 3. 誤差の分類（第2週）

::: tip 対策ポイント
各要因がどの誤差に該当するか判断できるようにする：
- **(A) 人為的誤差**：操作ミス、設定誤り、記録ミスなど
- **(B) 系統誤差**：離散化誤差、センサの個体差、オフセットなど
- **(C) 不定誤差**：環境変動（室温変化）、ゼロ点変動など
:::

### 4. 最小二乗法（第3〜4週）

::: tip 対策ポイント
- 評価関数 $J(a, b) = \sum (f(x_k) - y_k)^2$ を書ける
- 偏微分 = 0 の連立方程式を立てられる
- 近似関数の選び方（散布図から判断、理論式があれば利用）
:::

### 5. ヒステリシス（第7週）

::: tip 対策ポイント
- 「入力の増大・減少で出力が異なる」を正しく説明できる
- ヒステリシスが大きいセンサの注意点を述べられる
:::

### 6. 移動平均（第6週）

::: tip 対策ポイント
- 3点・5点単純平均法による平滑化計算ができる
- 端点の処理（0で補外）を理解する
- リアルタイム計測での移動平均の問題点（遅れ）を説明できる
:::

**練習問題**：次のデータを3点単純平均法で平滑化せよ（端点は0.0で補外）

| x [秒] | 0.0 | 2.0 | 4.0 | 6.0 | 8.0 |
|--------|-----|-----|-----|-----|-----|
| y | 1.0 | 2.0 | 5.0 | 6.0 | 3.0 |

<details>
<summary>解答</summary>

- $x = 0.0$：$(0.0 + 1.0 + 2.0) / 3 = 1.0$
- $x = 2.0$：$(1.0 + 2.0 + 5.0) / 3 = 2.7$
- $x = 4.0$：$(2.0 + 5.0 + 6.0) / 3 = 4.3$
- $x = 6.0$：$(5.0 + 6.0 + 3.0) / 3 = 4.7$
- $x = 8.0$：$(6.0 + 3.0 + 0.0) / 3 = 3.0$

</details>

### 7. 誤差伝播（第2〜3週）

::: tip 対策ポイント
- 積の誤差伝播：$\epsilon = M_1 \epsilon_2 + M_2 \epsilon_1$
- 有効数字を適切に処理する
:::
