## 3. ロボットとセンサ

### 3.1 🤔 もし感覚を失ったら？

まず、こんな場面を想像してみてください：

> **目を閉じて、厚い手袋をはめて、生卵を割らずにお椀に入れられますか？**

答えは「とても難しい」です。なぜなら：
- **視覚**がないから卵の位置がわからない
- **触覚**がないから力加減ができない
- **フィードバック**（修正）ができない

::: warning 💡 重要な気づき
感覚がなければ、精密な動作は不可能！

**ロボットも同じ**です。センサがなければ「目隠しで作業する」のと同じ状態になります。
:::

### 3.2 なぜセンサを学ぶのか？

ロボットが「見て」「感じて」「動く」ためには、外界の情報を取得する必要があります。

**センサ ＝ ロボットに「感覚」を与える装置**

| 人間の感覚器官 | ロボットのセンサ | 計測対象 |
|:-------------:|:---------------:|:--------:|
| 👁️ 目 | カメラ、LiDAR | 形状、距離、色 |
| 👂 耳 | マイク、超音波 | 音、距離 |
| 🖐️ 皮膚 | 力/圧力センサ | 力、接触 |
| ⚖️ 内耳（前庭） | IMU、ジャイロ | 姿勢、加速度 |
| 🌡️ 皮膚 | 温度センサ | 温度 |

::: tip 💡 ポイント
この授業では、ロボットの「感覚器官」であるセンサの原理と応用を学びます。
:::

## 4. センサとは何か

### 4.1 センサの定義

::: info 定義
**センサ（Sensor）** とは：

> A device which detects or measures a **physical property** and records, indicates, or otherwise **responds** to it.
> 
> — Oxford Advanced Learner's Dictionary

つまり、**Physical property（物理量）を検出または計測して、それを記録したり指し示したりする、もしくはそれに反応する device（装置・素子）** のこと。
:::

### 4.2 センサの分類

```mermaid
graph TD
    A[センサ] --> B[内界センサ<br>Internal Sensors]
    A --> C[外界センサ<br>External Sensors]
    
    B --> D[関節角センサ]
    B --> E[角速度センサ]
    B --> F[トルクセンサ]
    
    C --> G[視覚センサ]
    C --> H[距離センサ]
    C --> I[力センサ]
    C --> J[触覚センサ]
```

#### 内界センサ（Internal Sensors）

ロボットや機械システムの**内部の情報**を計測するセンサ

- 関節角（Joint angle）
- 関節角速度（Angular velocity）
- モータ電流・トルク

#### 外界センサ（External Sensors）

ロボットの**周辺環境の情報**を計測するセンサ

- 視覚（Vision）
- 距離（Range）
- 力（Force）
- 触覚（Tactile）

## 5. センサ＝感覚器

### 5.1 人間の五感との対比

ロボットにとってのセンサ ＝ 人間にとっての感覚器（五感）

| 人間の感覚 | 対応するセンサ例 |
|-----------|-----------------|
| 視覚 👁️ | カメラ、光センサ |
| 聴覚 👂 | マイク、超音波センサ |
| 触覚 ✋ | 力センサ、圧力センサ |
| 嗅覚 👃 | ガスセンサ |
| 味覚 👅 | 化学センサ |


### 5.2 🧠 脳の感覚マッピング（ホムンクルス）

人間の脳では、体の各部位の感覚・運動に割り当てられる皮質の面積が均等ではありません。

![Homunculus（脳地図）：感覚野と運動野の体部位マッピング](/images/homunculus.png)

*出典: Penfield & Rasmussen (1950) "The Cerebral Cortex of Man" - 教育目的で引用*

**なぜ手と口が大きい？**
- **手**：細かい操作が必要 → 神経の密度が高い → 高分解能
- **唇・舌**：繊細な感覚 → 高感度
- **背中**：大まかな感覚でOK → 低密度

::: tip 🔧 工学的な意味
ロボット設計でも同じ考え方を適用します：

| 部位 | 要求精度 | センサ配置 |
|:----:|:-------:|:----------:|
| ロボットハンドの指先 | 高精度 | 高密度・高分解能センサ |
| アームの本体 | 中程度 | 位置センサのみ |
| 台座 | 低精度 | 衝突検知程度 |

**センサの「分解能」と「密度」を用途に応じて配置する**ことが重要です。
:::

- **放射線**（Radiation）
- **磁気**（Magnetism）
- **超音波**（Ultrasonic）
- **近接覚**（Proximity）

## 6. センサは物理量の変換装置

### 6.1 物理量変換の原理

センサの本質は**物理量の変換（Translation of physical properties）** です。

```mermaid
graph LR
    A[物理量<br>Physical Properties] --> B[センサ<br>Sensor]
    B --> C[電気信号<br>Electric Signals]
    
    style B fill:#f9f,stroke:#333,stroke-width:2px
```

### 6.2 入力となる物理量

- 力（Force）
- 距離（Range）
- 速度（Velocity）
- 温度（Temperature）
- 圧力（Pressure）

### 6.3 出力となる物理量

- 電圧（Voltage）
- 抵抗（Resistance）
- インダクタンス（Inductance）
- ひずみ（Strain）
- 磁場（Magnetic field）

### 6.4 変換の具体例

| 入力物理量 | 変換後 | センサ例 |
|-----------|--------|---------|
| 力（Force） | 電圧 | ひずみゲージ |
| 距離（Range） | 電圧/パルス | 測距センサ |
| 温度（Temperature） | 抵抗値 | サーミスタ |
| 回転角（Angle） | パルス | エンコーダ |

## 7. センサの動作メカニズム

### 7.1 一般的なセンサの働き

センサの動作は以下の4ステップで構成されます：

```mermaid
graph LR
    A["(1) 物理量入力<br>Physical Quantity"] --> B["(2) 検出<br>Detection"]
    B --> C["(3) 信号変換<br>Signal Translation"]
    C --> D["(4) 電気信号出力<br>Electric Signals"]
    
    subgraph Sensor
        B
        C
    end
```

1. **物理量が入力される** — 力、温度、距離など
2. **センサ素子がそれを検出** — Sensing elements
3. **何かしらの信号に変換する** — Transducer
4. **電気的信号が出力される** — 電圧、電流、抵抗変化など

## 8. メカトロニクスにおけるセンサ

### 8.1 フィードバック制御

センサはフィードバック制御システムの重要な構成要素です。

```mermaid
graph LR
    A[入力<br>Input] --> B[コントローラ<br>Controllers]
    B --> C[アクチュエータ<br>Actuators/Motors]
    C --> D[ロボット<br>Robot]
    D --> E[出力<br>Output]
    D --> F[センサ<br>Sensors]
    F -->|フィードバック| B
    
    style F fill:#f9f,stroke:#333,stroke-width:2px
```

### 8.2 制御ループの流れ

1. **目標値（入力）** を設定
2. **コントローラ** が指令値を計算
3. **アクチュエータ（モータ）** が動作
4. **ロボット** が動く
5. **センサ** が現在状態を計測
6. 目標値との差を計算 → 2に戻る（フィードバック）

::: warning 注意
フィードバック制御の詳細は第6-7週で学びます。
ここでは概念だけ理解しておきましょう。
:::

## 9. 人間 vs ロボット：制御システムの比較

人間の感覚制御とロボットのセンサ制御を対比すると、驚くほど似ていることがわかります：

| 比較項目 | 人間 | ロボット | 計測工学の課題 |
|:--------:|:----:|:--------:|:--------------:|
| **目標設定** | 脳の意図（「水を飲みたい」） | 設定値 Setpoint（座標、力の値） | — |
| **制御中枢** | 大脳皮質 | マイコン / CPU | 信号処理 |
| **信号伝達** | 神経系（生体電気） | 電線 / バス（電圧、デジタル信号） | A/D変換、ノイズ対策 |
| **実行機構** | 筋肉・骨格 | モータ・シリンダ | — |
| **感知（フィードバック）** | 感覚器官（👁️目、🖐️皮膚） | センサ（📷カメラ、⚖️力覚） | **精度・感度・応答性** |

::: tip 💡 計測工学の使命
センサが取得する信号は、ノイズが含まれていたり、非線形だったりします（人間でも強い光で眩しく感じるように）。

**計測工学** ＝ ロボットの「感覚器官」を**正確・高速・安定**に動作させる技術

この授業を通じて、センサの原理と信号処理の基礎を学びましょう。
:::

## 📝 確認問題

### Q1. センサの役割として最も適切な説明は？

- [ ] A. 電力をロボットに供給する
- [ ] B. ロボットの動きを決定する
- [x] C. 物理量を検出し、電気信号に変換する
- [ ] D. プログラムを保存する

::: details 解説
センサは「物理量（力、距離、温度など）を検出・計測して電気信号に変換する装置」です。人間の感覚器官に相当します。
:::

### Q2. 厚い手袋をして目を閉じると、生卵を割らずにお椀に入れるのが難しくなります。この理由として正しいものは？

- [ ] A. 筋肉が弱くなるから
- [ ] B. 体温が下がるから
- [x] C. 視覚と触覚のフィードバックがなくなるから
- [ ] D. 手袋が重いから

::: details 解説
感覚（センサ）がないと、自分の動作の結果を確認できず、誤差を修正できません。これが「フィードバックがない」状態です。ロボットも同様で、センサがなければ精密な制御は不可能です。
:::

### Q3. 人間の「内耳（前庭器官）」に対応するロボットのセンサは？

- [ ] A. カメラ
- [ ] B. マイク
- [x] C. IMU（慣性計測装置）/ ジャイロセンサ
- [ ] D. 温度センサ

::: details 解説
内耳の前庭器官は平衡感覚（姿勢・加速度）を検知します。ロボットではIMU（Inertial Measurement Unit）やジャイロセンサが同じ役割を果たします。
:::

### Q4. 人脳感覚マッピング（ホムンクルス）で手や唇が大きく描かれている理由は？

- [ ] A. 実際に手や唇が大きいから
- [x] B. 神経の密度が高く、分解能が高いから
- [ ] C. よく動かすから
- [ ] D. 血流が多いから

::: details 解説
ホムンクルスで手や唇が大きいのは、その部位に対応する脳の皮質面積が大きい（＝神経密度が高い）からです。これは「高い分解能」を意味し、ロボット設計でも同様に、精密な操作が必要な部位には高精度センサを配置します。
:::

### Q5. フィードバック制御システムにおいて、センサの役割は？

- [ ] A. 目標値（Setpoint）を設定する
- [ ] B. モータに電力を供給する
- [x] C. 出力（実際の動き）を計測し、コントローラに返す
- [ ] D. 指令値を計算する

::: details 解説
フィードバック制御では：Input → Controller → Actuator → Robot → Output の流れがあり、**センサ**はOutputを計測してControllerに戻します。これにより目標との誤差を修正できます。
:::

### Q6. 次のうち「内界センサ」に分類されるものは？

- [x] A. 関節角センサ（エンコーダ）
- [ ] B. カメラ
- [ ] C. 距離センサ（LiDAR）
- [ ] D. 触覚センサ

::: details 解説
- **内界センサ**：ロボット内部の状態を計測（関節角、角速度、モータ電流など）
- **外界センサ**：ロボット外部の環境を計測（視覚、距離、力、触覚など）

エンコーダは関節の回転角を計測する内界センサです。
:::

### Q7. センサがない場合の制御を何と呼ぶ？

- [x] A. 開ループ制御（Open-loop control）
- [ ] B. 閉ループ制御（Closed-loop control）
- [ ] C. フィードバック制御
- [ ] D. 適応制御

::: details 解説
- **開ループ制御**：センサなし、フィードバックなし → 誤差を修正できない
- **閉ループ制御**：センサあり、フィードバックあり → 誤差を修正できる

センサは閉ループ制御を実現するために不可欠です。
:::
